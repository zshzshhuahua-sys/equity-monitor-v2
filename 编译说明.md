## 三、Dockerfile 详解

### 3.1 文件位置

```bash
./Dockerfile
```

### 3.2 内容解析

```dockerfile
FROM python:3.12-slim                    # 基础镜像

WORKDIR /app                              # 工作目录

RUN apt-get update && apt-get install -y \ # 安装系统依赖
    curl \
    && rm -rf /var/lib/apt/lists/*

COPY requirements.txt .                    # 复制依赖文件

RUN pip install --no-cache-dir -r requirements.txt  # 安装Python依赖

COPY src ./src                             # 复制应用代码
COPY web ./web
COPY config ./config
COPY templates ./templates
COPY data ./data

RUN mkdir -p /app/data                     # 创建数据目录

COPY src/main.py /app/                    # 复制入口文件

EXPOSE 8000                                # 暴露端口

ENV PROJECT_ROOT=/app                      # 环境变量

CMD ["sh", "-c", "python -m uvicorn src.main:app --host 0.0.0.0 --port 8000"]
```

### 3.3 关键组件说明

| 组件 | 版本 | 说明 |
|------|------|------|
| python:3.12-slim | 3.12 | 轻量级Python镜像 |
| fastapi | 0.115.0 | Web框架 |
| uvicorn | 0.32.0 | ASGI服务器 |
| sqlalchemy | 2.0.36 | ORM框架 |
| aiosqlite | 0.20.0 | 异步SQLite驱动 |
| akshare | >=1.15.0 | 财经数据API |

## 四、docker-compose.yml 详解

### 4.1 文件位置

```bash
./docker-compose.yml
```

### 4.2 内容解析

```yaml
version: '3.8'

services:
  equity-monitor:
    build: .                          # 构建镜像
    container_name: equity-monitor   # 容器名称
    ports:
      - "8001:8000"                 # 端口映射
    volumes:
      - ./data:/app/data            # 数据目录映射
      - ./config:/app/config        # 配置目录映射
    environment:
      - PROJECT_ROOT=/app           # 环境变量
    restart: unless-stopped          # 自动重启
    healthcheck:                    # 健康检查
      test: ["CMD", "curl", "-f", "http://localhost:8000/health"]
      interval: 30s
      timeout: 10s
      retries: 3
```

### 4.3 配置说明

| 配置项 | 说明 |
|--------|------|
| build | 指定Dockerfile位置 |
| container_name | 容器名称 |
| ports | 端口映射 (宿主机:容器) |
| volumes | 数据卷映射 |
| environment | 环境变量 |
| restart | 重启策略 |
| healthcheck | 健康检查 |

## 六、构建优化

### 6.1 加速构建

```bash
# 使用国内镜像源
docker build --build-arg PYTHON_INDEX_URL=https://mirrors.aliyun.com/pypi/simple/ .
```

### 6.2 减小镜像体积

```dockerfile
# 多阶段构建示例
FROM python:3.12-slim as builder
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

FROM python:3.12-slim
WORKDIR /app
COPY --from=builder /usr/local/lib/python3.12/site-packages /usr/local/lib/python3.12/site-packages
COPY . .
```

## 八、本地开发模式

### 8.1 不使用Docker

```bash
# 创建虚拟环境
python3.12 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 设置环境变量
export PROJECT_ROOT=/path/to/project

# 启动服务
python src/main.py
# 或
uvicorn src.main:app --reload --port 8001
```

### 8.2 使用Docker开发

```bash
# 挂载代码卷，实现热更新
docker run -d \
  -p 8001:8000 \
  -v $(pwd)/src:/app/src \
  -v $(pwd)/web:/app/web \
  --name equity-dev \
  equity-monitor
```
