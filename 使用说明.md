# 股权激励监控面板 V2.0 - Docker 部署完整说明

## 一、系统概述

基于 FastAPI + Vue 的 A股股权激励监控面板，支持实时价格监控、溢价率计算、预警通知。

### 功能特性

- ✅ 实时价格监控（腾讯财经API）
- ✅ 溢价率计算与三色预警（绿<80% 黄80%-150% 红>=150%）
- ✅ 手工添加/导入股票
- ✅ CSV/Excel批量导入
- ✅ 监控列表管理（编辑、删除）
- ✅ 数据持久化（SQLite）
- ✅ 支持北京时间自动判断交易时间

### 技术栈

| 组件 | 说明 |
|------|------|
| 后端 | FastAPI + Python 3.12 |
| 数据库 | SQLite (文件存储) |
| 前端 | HTML + TailwindCSS + Vanilla JS |
| 部署 | Docker + OrbStack |

## 三、本地重新部署

如果需要重新部署，按以下步骤操作：

### 3.1 环境要求

- Docker Desktop 或 OrbStack
- 端口 8001 未被占用

### 3.2 部署命令

```bash
# 1. 进入项目目录
cd ~/Desktop/股权激励监控面板_Docker部署

# 2. 构建镜像
docker build -t equity-monitor .

# 3. 运行容器
docker run -d \
  --name equity-monitor \
  -p 8001:8000 \
  -v $(pwd)/data:/app/data \
  -v $(pwd)/config:/app/config \
  -e PROJECT_ROOT=/app \
  --restart unless-stopped \
  equity-monitor
```

### 3.3 使用 docker-compose（推荐）

```bash
# 一键启动
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

### 4.2 添加股票

**方式一：手工添加**
1. 点击「➕ 手工添加」按钮
2. 输入股票代码（如 600519）
3. 输入行权价格
4. 点击「添加」

**方式二：CSV导入**
1. 点击「📄 模板」下载模板
2. 编辑模板，填写股票信息
3. 点击「📥 导入」上传文件

### 4.3 编辑/删除

1. 点击「✏️ 编辑」进入编辑模式
2. 点击股票名称可编辑该股票信息
3. 双击股票代码可删除该股票
4. 选中复选框后可批量删除

### 4.4 监控设置

- 点击「▶️ 启动监控」开启自动价格获取
- 轮询间隔：5分钟（可在 config/settings.yaml 修改）
- 溢价率计算：最新价 / 行权价 × 100%

## 五、配置说明

### 5.1 配置文件位置

```
./config/settings.yaml
```

### 5.2 配置项说明

```yaml
monitor:
  interval_seconds: 300      # 轮询间隔（秒），默认5分钟
  trading_hours_only: false # 是否只在交易时间获取价格
  enabled: true             # 是否启用监控

alert:
  enabled: true            # 是否启用预警
  cooldown_minutes: 30      # 预警冷却时间（分钟）
  thresholds:
    watch: 0.05           # 关注阈值（5%）
    warning: 0.10         # 警告阈值（10%）
    critical: 0.20        # 严重阈值（20%）

app_name: 股权激励监控面板
port: 8000
host: 0.0.0.0
```

## 七、常见问题

### Q1: 容器启动失败？

```bash
# 查看日志
docker logs equity-monitor

# 常见错误：
# 1. 端口被占用 → 修改端口映射 -p 8002:8000
# 2. 数据目录权限问题 → chmod -R 777 data
```

### Q2: 价格获取失败？

- 检查网络连接
- 腾讯财经API可能在海外无法访问
- 可配置代理或使用其他数据源

### Q3: 如何更新版本？

```bash
# 1. 停止容器
docker-compose down

# 2. 重新构建
docker-compose build

# 3. 启动
docker-compose up -d
```
