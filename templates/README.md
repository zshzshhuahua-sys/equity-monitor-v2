# 导入模板说明

## CSV 模板 (import_template.csv)

可直接使用文本编辑器编辑，或导入 Excel 后另存为 CSV。

### 字段说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| symbol | 是 | 6位股票代码 | 000001 |
| exchange | 是 | 交易所代码 (SH/SZ/BJ) | SZ |
| strike_price | 是 | 执行价格 (元) | 15.50 |
| quantity | 否 | 持有数量 (股) | 1000 |
| custom_threshold | 否 | 自定义预警阈值 (%，留空使用全局设置) | 15 |
| notes | 否 | 备注信息 | 备注内容 |

### 交易所代码说明

- **SH** - 上海证券交易所 (主板 600/601/603, 科创板 688)
- **SZ** - 深圳证券交易所 (主板 000, 创业板 300)
- **BJ** - 北京证券交易所 (430/83/87/88)

## Excel 模板格式

如果使用 Excel 文件 (.xlsx)，请保持相同的表头和字段顺序。

### 示例数据

| symbol | exchange | strike_price | quantity | custom_threshold | notes |
|--------|----------|--------------|----------|------------------|-------|
| 000001 | SZ | 15.50 | 1000 | | |
| 600519 | SH | 1200.00 | 500 | 25 | |
| 300750 | SZ | 45.80 | 2000 | 15 | 创业板股票 |
| 688111 | SH | 35.60 | 800 | 20 | 科创板 |

## 注意事项

1. **编码**: CSV 文件请使用 UTF-8 编码
2. **数字**: 执行价格和数量请使用数字格式，避免添加货币符号或千分位
3. **阈值**: 自定义阈值填写百分比数字，如 15 表示 15%，留空则使用全局默认阈值
4. **重复导入**: 相同股票代码再次导入时会自动更新已有记录
